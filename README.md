# 🚀 Smart Vault - Intelligent Backup & Disaster Recovery System

## 📌 Overview
Smart Vault is an **automated, cost-efficient, and intelligent** backup system for AWS EC2 instances, ensuring business continuity and regulatory compliance.

## 🔹 Features
✔ **Automated EC2 Backups using EBS Snapshots**  
✔ **Backup Scheduling with AWS EventBridge**  
✔ **Real-time Monitoring via CloudWatch**  
✔ **Failure Notifications with AWS SNS**  
✔ **Web UI for Manual Backup & Status Viewing**  
✔ **Disaster Recovery via S3 Cross-Region Backup**  

## 🛠️ Tech Stack
- **Backend:** FastAPI/Flask (Python)
- **Frontend:** React (JavaScript)
- **Cloud Services:** AWS Lambda, EBS Snapshots, S3, SNS, CloudWatch, API Gateway
- **Infrastructure:** Docker, Kubernetes (EKS), Terraform
- **Monitoring:** AWS CloudWatch, Prometheus, Grafana
- **Authentication (Optional):** AWS Cognito
